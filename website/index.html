[ MEMORY / INVARIANTS ]

Write nothing that cannot be checked.

  1. Raw memory is preserved unless explicitly forgotten.
  2. Every derived memory links to evidence.
  3. Current semantic facts have temporal validity.
  4. Contradictions are represented, not silently overwritten.
  5. Forgetting propagates to derived memory and indexes.
  6. Retrieval returns evidence-aware context packets.
  7. The daemon is the canonical writer.
  8. LLM output is validated before deterministic writes.
  9. Evaluation is a first-class package.
  10. The common path stays trivial: start, remember, retrieve.
